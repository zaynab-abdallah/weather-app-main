# تعليمات تشغيل المشروع - Next.js Weather App

## الخطوة 1: تثبيت Node.js
تأكد من تثبيت Node.js (النسخة 18 أو أحدث)
- تحميل من: https://nodejs.org/

## الخطوة 2: تثبيت المكتبات

افتح Terminal (PowerShell) في مجلد المشروع وقم بتشغيل:

```powershell
npm install
```

انتظر حتى يتم تثبيت جميع المكتبات المطلوبة.

## الخطوة 3: تشغيل المشروع

```powershell
npm run dev
```

## الخطوة 4: فتح المتصفح

افتح المتصفح وانتقل إلى:
```
http://localhost:3000
```

## المشاكل الشائعة

### مشكلة: الصور لا تظهر
**الحل:** تأكد من وجود مجلد `public/assets` بجميع الصور والخطوط.

### مشكلة: خطأ في التثبيت
**الحل:** احذف مجلد `node_modules` و ملف `package-lock.json` ثم قم بتشغيل `npm install` مرة أخرى.

### مشكلة: المنفذ 3000 مستخدم
**الحل:** استخدم منفذ آخر:
```powershell
npm run dev -- -p 3001
```

## البناء للإنتاج

```powershell
npm run build
npm start
```

## الفرق بين النسخة العادية و Next.js

### النسخة العادية (index.html):
- HTML, CSS, JavaScript مباشرة
- سهلة للمبتدئين
- لا تحتاج تثبيت

### نسخة Next.js:
- أداء أفضل
- SEO محسّن
- TypeScript للأمان
- Components منظمة
- سهولة الصيانة
- إمكانيات أكثر للتوسع

## اختيار النسخة المناسبة

- **للمبتدئين أو مشاريع بسيطة**: استخدم `index.html`
- **للمشاريع الاحترافية**: استخدم Next.js
